Edit

If you'd like to contribute some information to this website, fix a grammatical error or anything else, you can do so via our repository on github.

Simply login to github (or register for a free account), click on the file you wish to edit (e.g. for the call for proposals, click on cfp.md, or the supplementary files in the _cfp folder), then the pencil icon. Once you have saved your change you will need to click on 'create pull request' to send it to us. Don't worry about making a mistake, we'll review your edit before making it live.
